# GitHub Integration Guide

## Launching MARS 4.6
Open the **MARS 4.6** application. You will find a new **GitHub** tab in the menu bar that includes:

- **Sign In / Sign Out**
- **Upload**
- **Download**

### 1. Sign In
Before using GitHub features, sign in through the GitHub tab using your GitHub account credentials.

---

## 2. Uploading Files
To upload a file to GitHub:

1. Open the **Upload** option under the GitHub tab.  
2. Select the file on your local machine.  
3. Choose an existing repository or create a new one through the interface.  
4. Enter a branch name (e.g., `main`) and a commit message.  
5. Click **Submit**.  

You can verify the upload by checking your repository on GitHub.

---

## 3. Downloading Files
To download a file from GitHub:

1. Open the **Download** option under the GitHub tab.  
2. Select your repository and the desired file.  
3. Save the file locally when prompted.  
4. The downloaded file will open in a new tab inside MARS for editing or execution.
